Career Roadmap

AI / ML Engineer

A structured, stage-by-stage path from zero to production-ready

⚡ Where to Start — The Order That Works

01Python Basics
→
02Math Foundations
→
03Python for Data Science
→
04ML/AI Concepts
→
05Deep Learning & Specialization

Why this order? Python gives you a tool to experiment immediately. Math runs in parallel from Stage 2 — don't wait until it's "done" (it never is). ML concepts only click once you can code them and understand the math behind them. Learn math just enough before each ML topic, not all upfront.

Phase 01 · Beginner
Python Core + Math Foundations
~3–5 months
🐍
Python — Core Language
Essential — Learn All of This
  • Variables, data types, operators
  • Strings & string methods
  • Lists, tuples, sets, dicts
  • Conditionals & loops
  • Functions & scope
  • File I/O & error handling
  • Modules & packages (pip)
  • List/dict comprehensions
  • Basic OOP (classes, methods)
Python.org → Kaggle Python → Real Python
Official docs to start → Kaggle for practice → Real Python for deeper concepts
📐
Linear Algebra
Essential — Key Topics Only
  • Vectors & vector operations core
  • Matrices & matrix multiplication core
  • Dot products & norms core
  • Transpose, inverse core
  • Eigenvalues & eigenvectors core
  • SVD (Singular Value Decomp.) core
  • Span, basis, rank useful
  • Advanced proofs skip early
Khan Academy → 3Blue1Brown → StatQuest
Mechanics → Visual intuition → ML application
📊
Statistics & Probability
Essential — Key Topics Only
  • Mean, median, mode, variance, std core
  • Probability rules & Bayes' Theorem core
  • Distributions (Normal, Binomial, etc.) core
  • Hypothesis testing & p-values core
  • Correlation & covariance core
  • Confidence intervals useful
  • Bayesian inference useful
  • ANOVA, Chi-squared tests defer
Khan Academy → StatQuest
Khan Academy is strongest here — rely on it heavily. StatQuest connects each concept to ML.
∫
Calculus
Yes — You Need It for Deep Learning
  • Derivatives & differentiation rules core
  • Chain rule core
  • Partial derivatives core
  • Gradients & gradient descent core
  • Integrals (basic understanding) useful
  • Multivariable calculus (deep) defer
Khan Academy → 3Blue1Brown
3Blue1Brown's "Essence of Calculus" is unmissable for gradient intuition.
Phase 02 · Intermediate
Python for Data Science + Classical ML
~4–6 months
🔢
NumPy
Essential
  • Arrays, shapes, dtypes
  • Broadcasting & vectorized ops
  • Indexing & slicing
  • Linear algebra ops (np.linalg)
  • Random number generation
NumPy Docs → Kaggle
Official docs are excellent. Kaggle notebooks for hands-on practice.
🐼
Pandas
Essential
  • DataFrames & Series
  • Loading CSV, JSON, Excel
  • Filtering, sorting, groupby
  • Handling missing data
  • Merging & joining datasets
  • Apply, map, lambda
Pandas Docs → Kaggle Pandas
Kaggle's free Pandas course is one of the best short courses available.
📈
Matplotlib & Seaborn
Essential
  • Line, bar, scatter, histogram plots
  • Customizing axes, titles, legends
  • Subplots & figure layouts
  • Heatmaps, pairplots (Seaborn)
  • Plotly for interactive charts optional
Matplotlib Docs → Kaggle DataViz
Kaggle's Data Visualization course covers both Matplotlib and Seaborn well.
🔬
Scikit-learn
Essential
  • Train/test split, cross-validation
  • Preprocessing & pipelines
  • Linear & Logistic Regression
  • Decision Trees, Random Forests
  • SVM, KNN, Naive Bayes
  • Clustering (K-Means, DBSCAN)
  • Metrics: accuracy, F1, AUC-ROC
Sklearn Docs → Kaggle ML → StatQuest
Scikit-learn docs + Kaggle to practice, StatQuest to understand what's happening under the hood.
🧮
Discrete Math
Partial — Key Topics Only
  • Sets, relations, functions core
  • Boolean logic & truth tables core
  • Graph theory basics useful
  • Combinatorics & counting useful
  • Formal proofs & logic skip
Khan Academy → Coursera
Only study the core topics listed above — don't go deep into formal proofs.
🗄️
SQL
Essential — Real World Data Lives Here
  • SELECT, WHERE, ORDER BY
  • JOIN (inner, left, right)
  • GROUP BY & aggregations
  • Subqueries & CTEs
  • Window functions useful
Kaggle SQL → SQLZoo → Mode SQL
Kaggle to learn, SQLZoo to drill, Mode for advanced queries.
Phase 03 · Advanced
Deep Learning + Specialization
~5–8 months
🧠
Deep Learning Foundations
Essential
  • Neural networks & activation functions
  • Backpropagation & gradient descent
  • Loss functions & optimizers (Adam, SGD)
  • Overfitting, dropout, regularization
  • Batch norm & learning rate tuning
Andrew Ng → fast.ai
Andrew Ng for theory first, fast.ai to build real things fast.
🔥
PyTorch or TensorFlow/Keras
Pick One — PyTorch Recommended
  • Tensors & autograd
  • Building custom models (nn.Module)
  • Training loops & dataloaders
  • GPU acceleration (CUDA)
  • Saving/loading models
PyTorch Docs → learnpytorch.io
learnpytorch.io (by Daniel Bourke) is the best free PyTorch course available.
🖼️
Computer Vision
Specialization Track
  • CNNs (Conv, Pool, Flatten layers)
  • Transfer learning (ResNet, EfficientNet)
  • Object detection (YOLO, Faster-RCNN)
  • Image segmentation
  • OpenCV basics optional
fast.ai → CS231n Stanford
fast.ai is practical and fast. CS231n is the gold standard deep dive for CV.
💬
NLP & LLMs
Specialization Track
  • Tokenization, embeddings, TF-IDF
  • RNNs, LSTMs (understand, not master)
  • Transformers & attention mechanism
  • BERT, GPT — fine-tuning with HuggingFace
  • Prompt engineering & RAG systems
HuggingFace NLP → CS224n Stanford
HuggingFace course is free and modern. CS224n for deeper NLP theory.
📉
Advanced Math
Learn As Needed
  • Information Theory (entropy, KL divergence) core
  • Optimization theory core
  • Lagrange multipliers useful
  • Fourier transforms defer
  • Stochastic processes defer
StatQuest → CS229 Stanford
Learn these topics as they come up in ML — not all at once upfront.
⚙️
Advanced Python
Essential at This Stage
  • Decorators & generators
  • Async programming (asyncio)
  • Type hints & dataclasses
  • Virtual envs & dependency mgmt
  • Writing clean, modular ML code
Real Python → Python Docs
Real Python has the best articles on advanced Python patterns for production code.

Git & Software Practices
Essential from Day One

Portfolio Projects
Your Ticket to Employment
  • End-to-end ML project (data → deploy)
  • Kaggle competition (top 20%)
  • NLP app or CV project
  • Research paper implementation
  • Open source contribution
Kaggle Competitions → Papers With Code
Kaggle for competitions & datasets. Papers With Code to find papers worth implementing.

🌐 Other Languages — Do You Need Them?

SQL essential
Real-world data lives in databases. Non-negotiable for any data role.
R skip
Only needed for academic/research stats roles. Python covers everything.
C++ advanced only
Needed for low-level ML infra, robotics, or custom CUDA kernels. Defer.
Bash/Linux useful
You'll live in the terminal. Learn basic scripting and navigation early.
Julia skip
Niche. Only relevant for scientific computing research.
